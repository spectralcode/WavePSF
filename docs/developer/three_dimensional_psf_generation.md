# 3D PSF Generation

How to implement 3D PSF generation in WavePSF? Let's check what is out there in the open-source space.

## Open-Source 3D PSF Generation Implementations

| Name | Backend | Method for PSF Generation | Links |
|---|---|---|---|
| **psf_generator** | PyTorch (CPU/GPU) | General microscope PSF simulation library; Cartesian Fourier propagators plus spherical/Bessel-style propagators. The Cartesian path includes custom zoomed-FFT / CZT utilities. | [README](https://github.com/Biomedical-Imaging-Group/psf_generator)<br>[propagators tree](https://github.com/Biomedical-Imaging-Group/psf_generator/tree/main/src/psf_generator/propagators)<br>[cartesian_propagator.py](https://github.com/Biomedical-Imaging-Group/psf_generator/blob/main/src/psf_generator/propagators/cartesian_propagator.py)<br>[czt.py](https://github.com/Biomedical-Imaging-Group/psf_generator/blob/main/src/psf_generator/utils/czt.py) |
| **PSFGenerator (Java / Fiji / Icy)** | Java | 3D diffraction-model package with Born & Wolf, Gibson & Lanni, Richards & Wolf, and variable-RI Gibson & Lanni models; uses adaptive numerical integration in the diffraction code. | [README](https://github.com/Biomedical-Imaging-Group/PSFGenerator)<br>[PSF.java](https://github.com/Biomedical-Imaging-Group/PSFGenerator/blob/master/src/psf/PSF.java)<br>[RichardsWolfPSF.java](https://github.com/Biomedical-Imaging-Group/PSFGenerator/blob/master/src/psf/richardswolf/RichardsWolfPSF.java)<br>[KirchhoffDiffractionSimpson.java](https://github.com/Biomedical-Imaging-Group/PSFGenerator/blob/master/src/psf/richardswolf/KirchhoffDiffractionSimpson.java) |
| **PSFToolbox-Matlab** | MATLAB + DipImage | Toolbox centered on `GenericPSFSim`; includes Richards-Wolff, SlicePropagation, RWFast, and CZT-based methods, with explicit voxel sampling via `ImageParam.Sampling = [dx dy dz]`. | [README](https://github.com/bionanoimaging/PSFToolbox-Matlab)<br>[GenericPSFSim.m](https://github.com/bionanoimaging/PSFToolbox-Matlab/blob/main/Code/ElectricFields/GenericPSFSim.m)<br>[RWFastProp.m](https://github.com/bionanoimaging/PSFToolbox-Matlab/blob/main/Code/ElectricFields/RWFastProp.m)<br>[ZoomedPupilProp.m](https://github.com/bionanoimaging/PSFToolbox-Matlab/blob/main/Code/ElectricFields/ZoomedPupilProp.m) |
| **PSF Simulation App** | MATLAB | Interactive PSF simulation/visualization for single-molecule microscopy; includes CZT-based resampling utilities and coordinate-generation helpers for object-space sampling. | [README](https://github.com/schneidermc/psf-simulation-app)<br>[czt2.m](https://github.com/schneidermc/psf-simulation-app/blob/main/src/functions/czt2.m)<br>[czt2_fast.m](https://github.com/schneidermc/psf-simulation-app/blob/main/src/functions/czt2_fast.m)<br>[create_coord.m](https://github.com/schneidermc/psf-simulation-app/blob/main/src/functions/create_coord.m) |
| **PyFocus** | Python | Fully vectorial focused-field calculation after arbitrary phase masks; useful as a PSF-adjacent reference for direct physical-grid control and integral/quadrature-based field computation. | [README](https://github.com/fcaprile/PyFocus)<br>[sim.py](https://github.com/fcaprile/PyFocus/blob/main/src/PyFocus/sim.py)<br>[base.py](https://github.com/fcaprile/PyFocus/blob/main/src/PyFocus/model/focus_field_calculators/base.py) |
| **MicroscPSF-Matlab** | MATLAB | Fast approximation of the 3D Gibson-Lanni PSF model; radial/Bessel-series style solver with interpolation to Cartesian output. | [README](https://github.com/MicroscPSF/MicroscPSF-Matlab)<br>[MicroscPSF.m](https://github.com/MicroscPSF/MicroscPSF-Matlab/blob/master/MicroscPSF.m) |
| **MicroscPSF-Py** | Python | Python implementation of the fast MicroscPSF Gibson-Lanni solver. | [README](https://github.com/MicroscPSF/MicroscPSF-Py)<br>[examples.ipynb](https://github.com/MicroscPSF/MicroscPSF-Py/blob/master/examples.ipynb) |
| **MicroscPSF-Cpp** | C++ | C++ member of the MicroscPSF family for fast 3D PSF computation. Good reference if a native C++ implementation is important. | [repo](https://github.com/MicroscPSF/MicroscPSF-Cpp) |
| **PSFmodels** | Python wrapper + compiled C++ backend | Scalar and vectorial 3D microscope PSF models with an explicit sample-space API (`dxy`, `dz`, `zv`), useful as a reference for clean physical sampling inputs. | [README](https://github.com/tlambert03/PSFmodels)<br>[_core.py](https://github.com/tlambert03/PSFmodels/blob/main/src/psfmodels/_core.py) |
